# SOC Dashboard - React Frontend

Modern SOC (Security Operations Center) dashboard built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI (shadcn/ui style)
- **Charts**: Recharts
- **Animations**: Framer Motion
- **State Management**: Zustand
- **Routing**: React Router v6

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:3000`

### Build for Production

```bash
# Build to docs/ folder (for GitHub Pages)
npm run build

# Preview production build
npm run preview
```

## Project Structure

## Features

- **Dashboard Overview**: KPI cards, severity distribution, incident narrative
- **Alerts View**: Filterable alerts table with detail drawer
- **Events (SIEM)**: Event log viewer with source filtering
- **Devices (EDR)**: Device inventory with process tree & network map
- **Automations (SOAR)**: Playbook library and run simulator
- **Timeline**: Attack chain visualization
- **MITRE ATT&CK**: Technique coverage matrix
- **Intel**: IOC management
- **Reports**: Markdown report viewer
- **Settings**: Theme, notifications, data management

## Data Source

The dashboard reads static JSON/JSONL files from `dashboard_data/`:

- `summary.json` - Dashboard summary stats
- `alerts.jsonl` - Security alerts
- `events.jsonl` - Security events
- `edr_devices.json` - Device inventory
- `playbooks.json` - SOAR playbooks
- `iocs.json` - Threat intelligence
- And more...

These files are generated by the Python pipeline (`python run_pipeline.py`).

## Deployment

The app builds to the `docs/` folder which is served by GitHub Pages.

```bash
npm run build
git add docs/
git commit -m "Deploy"
git push
```
